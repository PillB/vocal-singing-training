# Phase 3 — UI/UX & Feature Design

## Preamble

Design a clean, motivating, beginner-friendly interface with two tracks, individual + structured modes, recording/review, metrics, and a 12-week dashboard.

## Design principles
- **Calm dark theme** with vocal (blue) vs singing (amber) accents for track identity.
- **Non-judgmental copy** — scores as compass, not talent grade.
- **Progressive disclosure** — list → exercise depth (steps, tips, audio, metrics).
- **Local privacy** — recordings never leave the device.
- **Mobile-first responsive** grid and sticky header.

## Screens
1. Home + dual tabs  
2. Exercise player  
3. Structured session banner (pause/resume/end)  
4. History / audit  
5. 12-week plan  

## Phase 3 retrospection
- Design implemented in `css/styles.css` + `index.html`.
