# Exercise Expansion Research Notes

## Vocal / speaking (Vinh Giang–aligned)

Public themes from Vinh Giang’s teaching (STAGE programs, talks, and posts) that shaped **advanced** exercises:

| Theme | Exercise ID | Practice focus |
|-------|-------------|----------------|
| Strategic pause for processing & authority | v10-power-pause | Intentional silence after key ideas |
| Pause replaces fillers | v11-kill-fillers | um/uh → breath + pause |
| Tonality / melodic speech | v12-melodic-speech | Variety without monotony |
| Volume as confidence signal | v13-volume-ladder | Controlled dynamics 1–5 |
| Pace variation for engagement | v14-pace-variation | Slow on points that matter |
| Gestures & body language | v15-gestures | Purposeful visual channel |
| Facial expressiveness | v16-facial-expression | Face–voice congruence |
| Concision method (receive→breathe→refine→share) | v17-strategic-concision | Think before answer |
| Storytelling peak emotion | v18-story-peak | Setup → peak → point |
| Authority cadence / landings | v19-authority-close | Downward statement endings |
| Energy match & charisma | v20-energy-match | Calibrate then lead ~10% |

## Singing (complementary pedagogy)

Beyond Live Music School homework (s1–s3), advanced drills draw on standard contemporary voice science & pedagogy:

| Theme | Exercise ID | Notes |
|-------|-------------|-------|
| Lip trills (SOVT) | s4 | Back pressure, easy phonation |
| Sirens / glides | s5 | Smooth range connection |
| Straw phonation | s6 | Classic SOVT efficiency |
| Humming resonance | s7 | Forward buzz / mask |
| Breath support (S → /A/) | s8 | Even air before tone |
| Single-note pitch match | s9 | Visual equalizer focus |
| Five-note scales | s10 | Mid-range coordination |
| Dynamic swells | s11 | Messa-di-voce style control |
| Easy onset | s12 | Balanced start (not slam/breathy) |
| Arpeggio match | s13 | Chord-tone accuracy |
| Staccato vs legato | s14 | Articulation contrast |
