# Pitch games research → product mapping

## References

| Product | What works | We adopted |
|---------|------------|------------|
| **SingStar** | Pitch as position vs target lane; color when in range | Green **hit highway** around target; voice color by zone |
| **Yousician** | Real-time pitch + points + game feel | **Score, combo, PERFECT/GOOD flashes** |
| **Singing Carrots** | Pitch on piano layout; gamified accuracy | **Keyboard strip** target/voice highlight |
| **Vocal Match (Theta)** | Hear note → sing match → next | **Note challenge**: lock 8 notes by holding in-lane ~0.8s |
| **Simply Sing** | XP / progress motivation | Session score + auto-fill accuracy metrics |

## Zones (cents)

- Perfect ≤ 15¢  
- Good ≤ 35¢ (in-lane)  
- Close ≤ 60¢  
- Off beyond  

## UX loop

Start practice → target in lane → sing → combo builds → hold to lock (challenge) → piano plays next → score → save maps to metrics.
